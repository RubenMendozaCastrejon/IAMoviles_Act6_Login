# Prompt
Crea una aplicación llamada "Investech" con una estructura de navegación por rutas (routes) y un diseño Clean UI minimalista (estilo similar a la imagen adjunta, con mucho espacio en blanco y acentos en azul #0095FF).
Estructura de Archivos:
- main.dart: Configura el MaterialApp, el tema visual (fuentes limpias, colores primarios) y las rutas: '/', '/login' y '/registro'.
- archivos/inicio_pagina.dart: Pantalla de bienvenida con:
  - Título grande y subtítulo descriptivo.
  - Una ilustración central (Image.network).
  - Dos botones tipo "píldora" (bordes redondeados al 30%): uno con borde negro (OutlinedButton) para "Login" y uno sólido azul para "Sign up".
  - archivos/formulario_login.dart y archivos/formulario_registro.dart:
  - Encabezado con el nombre de la acción.
  - Campos de texto (TextField) personalizados: la etiqueta (label) debe estar fuera y arriba del campo de texto, no dentro como hint.
  - Un botón de acción principal al final que ocupe todo el ancho.
  - Un enlace de texto al final ("¿No tienes cuenta? Sign up") para navegar entre pantallas.
